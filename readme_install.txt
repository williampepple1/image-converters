IMAGE CONVERTER
================

Welcome to Image Converter - your all-in-one solution for converting images between popular formats!

FEATURES:
---------
• Convert images between multiple formats:
  - JPEG/JPG: Perfect for photos with smallest file sizes
  - PNG: Lossless quality with transparency support
  - GIF: Classic format for simple graphics and animations
  - BMP: Windows bitmap for maximum compatibility
  - ICO: Windows icons with multiple sizes (16, 32, 48, 256px)
  - WebP: Modern web format with excellent compression*
  - TIFF: Professional print/scanning workflows*
  - HEIC/HEIF: iPhone photos (requires optional libraries)*
  - AVIF: Next-gen web format (requires optional libraries)*

• Batch Conversion: Convert multiple files at once
• Quality Control: Adjustable quality slider for lossy formats
• Drag & Drop: Simply drag files into the application
• Image Preview: See your images before converting
• Progress Tracking: Watch conversion progress in real-time
• Cancel Support: Stop batch conversions anytime

SYSTEM REQUIREMENTS:
--------------------
• Windows 10/11 (64-bit)
• 50 MB free disk space
• No additional software required

*Some formats require optional libraries to be installed separately.
